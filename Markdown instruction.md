## Списки
Чтобы добавить ненумерованные списки, необходимо пункты выделить звёздочкой (*). 
Например:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать.
Например:
1. Элемент 1
2. Элемент 2
3. Элемент 3
